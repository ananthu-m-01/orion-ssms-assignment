# SSMS Assessment - Orion Innovation

This repository contains the SQL Server Management Studio (SSMS) assessment submissions for the online training program conducted by **Orion Innovation**.

## Assessment Details

| #  | Topic | Description |
|----|-------|-------------|
| 1  | [AND, OR, LIKE & BETWEEN Operators](./Assessment-1/README.md) | Working with logical and comparison operators in SQL |
| 2  | [Basic SQL Concepts](./Assessment-2/README.md) | Fundamentals of SQL including SELECT, INSERT, UPDATE, DELETE |
| 3  | [Joins - Customer Order Management System](./Assessment-3/README.md) | Implementing various JOIN operations using a customer order management database |
| 4  | Operators - Library Management System | Using SQL operators with a library management system database |

## Technologies Used

- SQL Server Management Studio (SSMS)
- Microsoft SQL Server
